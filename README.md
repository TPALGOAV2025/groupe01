Algorithme TP

Ce processus pratique consiste à développer une application web avec Django (Python) .
L'objet principal est implémenté sans tenir compte de l'idée algorithmique qui apparaît dans le contexte de la fonction applicative.

Ce projet utilise HTML, CSS et JavaScript pour l'interface utilisateur et SQLite comme base de données.
L'application, une fois l'interface utilisateur développée, a été convertie en fichier exécutable (.exe) , 
permettant son lancement direct sous Windows sans installation préalable de Python ou Django.

Pour exécuter l'application, elle vous permettra de le faire distet d'utiliser le filtre application.exe.
